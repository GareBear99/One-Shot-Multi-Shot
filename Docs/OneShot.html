What One Shot is (plain English)

One Shot is a live UP/DOWN direction signal that updates every 5 seconds. You can place a small, fixed‑expiry trade in either direction (with or against the signal). After expiry, your result is graded and the system’s rolling win‑rates update.
Important: One Shot is a signal + scoreboard. It is not a guaranteed money machine. The “edge” comes from being selective: only trade when confidence is stable and the win‑rate windows are healthy.

What “locked” vs “live confidence” means

  • Locked direction (every 5s): this is the actual prediction that gets scored later.
  • Live confidence (every 1s): a “temperature gauge” that can rise/fall without rewriting the locked direction.
  • Stability (HIGH/MED/LOW): how much the confidence meter is wobbling.

Quick Start (3 steps)

Step 1: Watch the signal for 30–60 seconds.
Trade only if confidence is stable and win‑rates look healthy.
Step 2: Choose UP or DOWN, set stake (0–100), pick expiry (30s/60s/5m/15m).
Step 3: Place the trade. After expiry, it’s graded and the scoreboards update.

Trade selection gates

  • Stability gate: avoid LOW stability (confidence jumping around).
  • Score gate: require minimum win‑rate in BOTH short windows (1m/5m) and longer windows (1h/4h/12h).
  • Cooldown gate: after a loss streak, pause and wait for the meter to “re-stabilize”.

The math (bankroll-first)

Per-trade outcomes

Profit if correct = stake × payoutRate
Loss if wrong = -stake

Example (stake = $10, payoutRate = 0.80):
Correct → +$8 profit (receive $18 total)
Wrong → -$10

Expected Value (EV) per trade

EV = W × (S×0.8) − (1−W) × S

W = win rate (0.72 means 72%)
S = stake
0.8 = payout rate (80% profit)

Break-even win rate

W = 1/1.8 ≈ 55.56% (for 80% payout)

Example monthly math (matches your “cap at $10” rule)

Assume:
Bankroll = $50
Typical stake = $5 (cap $10)
Trades/day = 10
Days/month = 22
Win rate = 72%

EV/trade = 0.72×(5×0.8) − 0.28×5
= 2.88 − 1.40
= $1.48 per trade

Daily EV ≈ 10 × 1.48 = $14.80
Monthly EV ≈ 22 × 14.80 = $325.60
EV is not guaranteed profit. Streaks + execution limits still exist.

Safety rules

Hard limits
• Stake cap: $10
• Max trades/day default: 10
• Stop after: 2–3 consecutive losses or -20% bankroll
• Only trade when stability is HIGH/MED
Soft limits
• If win‑rate windows drop → pause
• If confidence flips rapidly → pause
• If spread/fees spike → pause
• If market becomes news‑driven → pause
Reality check: no system “guarantees” 90% wins. The goal is selectivity + strict loss controls.

FAQ

Q: Can One Shot learn and adapt when accuracy drops?
A: Yes, but updates must be controlled (enough samples, validation window, no chasing every loss).

Q: Can this trade real funds?
A: Later. Start with paper trading, log everything, then integrate execution on a compliant venue.

Q: Best default setup for a $50 bankroll?
A: Typical stake $5, cap $10, ≤10 trades/day, stop on a short loss streak.
